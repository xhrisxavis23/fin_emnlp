## 베이스라인 별 진입점

### `alphaagent/`
- 패키지 설치: `pip install -e .` (pyproject.toml)
- CLI: `alphaagent/app/cli.py`, `alphaagent/app/qlib_rd_loop/`
- 시나리오: `alphaagent/scenarios/qlib/` (`prompts_alphaagent.yaml`, `prompts_rdagent.yaml`)
- KDD'25 페이퍼 PDF 동봉: `alpha_agent.pdf`, `KDD2026_finance.pdf`

### `rdagent/`
- 패키지 설치: `pip install -e .` (pyproject.toml)
- 시나리오: `rdagent/scenarios/`
- AlphaAgent 가 RD-Agent 위에 빌드되었으므로 두 디렉토리는 **공통 backbone**

### `alphaforge/`
- 단일 스크립트 실행: `python train_AFF.py`, `python combine_AFF.py`, `python backtest_pred.py`
- 결과 확인 노트북: `exp_AFF_calc_result.ipynb` 외 4개
- GP / DSO / RL 변형도 함께 (`train_GP.py`, `train_DSO.py`, `train_RL.py`)

### `qlib_benchmarks/`
- workflow 실행: `qrun examples/benchmarks/{LightGBM,Linear,MLP,XGBoost,Transformer,GATs,...}/workflow_config_*_Alpha158{,_csi500}.yaml`
- 일괄 실행: `examples/run_all_model.py`

### `qlib_topkdropout/`
- 실행 (FaVOR `run/` 모듈 의존):
  ```bash
  cd /home/dgu/fin/revision/favor   # PYTHONPATH 기준
  MARKET=cn python ../baselines/qlib_topkdropout/run_pipeline_topkdropout_lgbm.py "<concept>" --outer-loop 5
  ```
- 또는 각 파일을 `revision/favor/` 안으로 복사 후 실행
