## 새로운 실험 / 모델 / 코드가 필요할 때

1. **항상 새롭게 작성**합니다.
2. 기존 코드를 활용해야 한다면 → **복사 후 참고만** 합니다. 원본은 그대로 둡니다.
   - 예: `AlphaAgent/stage2.py`를 변형해 쓰고 싶다 → `revision/stage2_v2.py` 같은 식으로 **복사본**을 새 위치에 만들어 수정합니다.
3. 새로 만드는 모든 산출물(코드, 노트북, 결과, 그림, 문서)은 **새 디렉토리** 또는 **명확히 구분된 새 파일**로 분리합니다.
4. 산출물은 가급적 `revision/` 또는 EMNLP 작업용 신규 디렉토리에 격납해 기존 트리와 섞이지 않도록 합니다.

- **자동 모드(Auto mode)가 켜져 있더라도, 심지어 `--dangerously-skip-permissions` (bypass permissions) 가 켜져 있더라도** 위 절대 규칙은 **항상 우선**합니다. 이 규칙을 자동화·권한 우회로 무력화할 수 없습니다.
